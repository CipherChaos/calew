# Work Breakdown Structure — Template

Decompose project into deliverable work packages aligned with agent phases.

```mermaid
mindmap
  root((Project Name))
    Foundation
      Charter
      Glossary
    Architecture
      C4 diagrams
      API contract
    Implementation
    Verification
    Release
```

Or use numbered tree:

```
1.0 Project Name
├── 1.1 Work package
│   └── 1.1.1 Task
```
